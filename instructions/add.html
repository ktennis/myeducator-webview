Pet Boarding Program Instructions

Instructions:

  • Make sure you include comments.
  • Write a program that prompts the user to enter the number of customers for your Pet Boarding company.
  • You do not need to store each customer in a list.
  • You must use classes.

Customer Class

Instance Variables:

  • cust_id (string)
  • first_name (string)
  • last_name (string)
  • address1 (string)
  • address2 (string)
  • city (string)
  • state (string)
  • zip (string)
  • balance (float)
  • cust_pet (Pet object)

Constructor should:

  • Receive first_name, last_name, address1, address2, city, state, and zip and store them
  • Initialize balance to 0.0
  • Initialize cust_pet to None
  • Calculate cust_id using gen_id method

gen_id()

  • Receives first_name, last_name, address1
  • Creates ID using first 3 letters of first and last name and first 5 of address1 (no spaces)

return_bill()

Returns a string like:
Customer greand2677e with name greg anderson owes $123.50 for charlie's stay from 10/01/2020 to 10/20/2020

make_payment()

  • Receives a float
  • Subtracts amount from balance

Class Variable:

  • company_name = "Critter Watch"

Pet Class

Instance Variables:

  • pet_name (string)
  • breed (string)
  • age (int)
  • appointment (Appointment object)

Constructor receives: pet_name, breed, age, owner (Customer object)

Note: Call self.appointment = Appointment(owner) inside constructor

Appointment Class

Instance Variables:

  • begin_date (date)
  • end_date (date)
  • day_rate (float)
  • total_days (int)
  • total_cost (float)
  • owner (Customer object)

Methods:

  • Constructor: Receives owner (Customer)
  • set_appointment(begin_date, end_date, day_rate):
    • Calls calc_days()
    • Updates total_days and total_cost
    • Updates owner's balance
  • calc_days():
    • Calculates total_days as (end_date - begin_date).days
    • If total_days ≤ 0, set to 1
    • Calculates total_cost = total_days Ã— day_rate

Additional Notes

  • A Customer can only have one pet
  • Only the latest boarding info is kept
  • Company Name: Critter Watch (class variable)

Implementation Requirements

  • Prompt user for customer data and instantiate Customer
  • Prompt for pet data and assign Pet to cust_pet on Customer
  • Prompt for appointment data and call set_appointment()
  • Print bill with return_bill()
  • Call make_payment() and print updated bill

Bonus (Extra 3 Points)

  • Implement pet attribute as a list
  • Implement appointment attribute as a list
